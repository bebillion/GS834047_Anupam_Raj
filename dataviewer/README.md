# Data Viewer App

## Instructions

### Running the App

To start the development server:
```sh
npm run dev
# or
yarn dev
```

Open your browser and navigate to `http://localhost:3000`.

### Running Tests

To run the tests:
```sh
npm test
# or
yarn test
```

## Achievements
